# PKU-AFMU

## Quickstart

> require `python3.7` or above

```shell
$ pip3 install virtualenv

$ virtualenv venv

$ source ./venv/bin/activate

$ pip3 install -r requirements.txt

$ ./dev_start.sh    # run locally
$ ./dev_start.sh -g # or run with gunicorn
```
